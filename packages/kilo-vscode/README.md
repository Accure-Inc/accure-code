# Accure Code

**Accure Code is the all-in-one agentic engineering assistant.**
Build, ship, and iterate faster with the most popular open source coding agent.

## Key Features

- **Code Generation:** Accure Code can generate code using natural language.
- **Inline Autocomplete:** Get intelligent code completions as you type, powered by AI.
- **Task Automation:** Accure Code can automate repetitive coding tasks to save time.
- **Automated Refactoring:** Accure Code can refactor and improve existing code efficiently.
- **MCP Server Marketplace**: Accure Code can easily find and use MCP servers to extend the agent capabilities.
- **Multi Mode**: Plan with Architect, Code with Coder, and Debug with Debugger, and make your own custom modes.
- **AccureIQx Workspace Integration**: Built-in support for AccureIQx workspace management with custom modes like `accureiqx_app_builder` and `accureiqx_debug`.

## Get Started

1. Open the Accure Code sidebar in your VS Code interface.
2. The extension will automatically inherit credentials and settings from your active AccureIQx workspace.
3. Select your desired mode and start coding!
