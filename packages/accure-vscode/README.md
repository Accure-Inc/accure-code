# Accure Code

**The ultimate autonomous software engineer in your editor.**
Code, plan, and automate at the speed of thought with the world's most versatile open-source coding agent.

## Core Capabilities

- **Autonomous Coding:** Convert natural language prompts into complete, production-ready features.
- **Instant Autocomplete:** Experience lightning-fast, context-aware code completions as you type.
- **Task Orchestration:** Delegate complex, repetitive tasks and terminal commands to the agent.
- **Smart Refactoring:** Modernize, optimize, and refactor existing codebases with ease.
- **MCP Ecosystem Integration:** Instantly discover and connect Model Context Protocol (MCP) servers to expand the agent's capabilities.
- **Multi-Mode Flexibility:** Plan with the Architect, build with the Coder, or troubleshoot with the Debugger—or configure your own custom modes.
- **AccureIQx Integration:** Native workspace management with pre-configured developer environments like `accureiqx_app_builder` and `accureiqx_debug`.

## Getting Started

1. Open the Accure Code sidebar in your VS Code workspace.
2. The extension will automatically inherit credentials and configuration from your active AccureIQx workspace.
3. Select a mode and let Accure Code build with you.
